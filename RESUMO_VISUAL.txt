🚀 COMEÇAR A USAR
════════════════════════════════════════════════════════════════════════════════

1️⃣  PREPARAR SUPABASE
   └─ Criar conta em supabase.com
   └─ Criar novo projeto
   └─ Copiar credenciais

2️⃣  CONFIGURAR .env
   └─ VITE_SUPABASE_URL=https://...
   └─ VITE_SUPABASE_ANON_KEY=...

3️⃣  INSTALAR
   └─ npm install

4️⃣  RODAR
   └─ npm run dev

5️⃣  ACESSAR
   └─ Loja: http://localhost:5173
   └─ Admin: http://localhost:5173/admin
